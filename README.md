# UniGradePointAverageCalculation
University Grade Point Average Calculator

**Projenin Amacı**</br>
2 farklı not sisteminde kullanıcın girdiği ders kredilerine göre kullanıcın not ortalamasını hesaplar. İstenilirse iki not sistemi birlikte de kullanılabilinir.<br/>

**Kullanılan Kütüphaneler**<br/>
ViewBinding<br/>
Glide<br/>
